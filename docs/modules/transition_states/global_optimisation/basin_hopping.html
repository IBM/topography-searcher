Basin hopping

The basin_hopping module contains the BasinHopping class which performs global optimisation of a given function

class topsearch.global_optimisation.basin_hopping.BasinHopping(ktn: type, potential: type, similarity: type, step_taking: type)

Description

Class to perform basin-hopping global optimisation algorithm as described in https://doi.org/10.1021/jp970984n One iteration is described by: 1) Perturb current position 2) Minimisation to get corresponding local minimum 3) Accept/reject based on Metropolis-like criterion

ktn

An instance of the KineticTransitionNetwork class that is used to store the unique minima we locate

Type:

class instance

potential

Object containing the function that we are aiming to optimise

Type:

class instance

similarity

Object that determines if two minima are the same or unique

Type:

class instance

step_taking

Instance that perturbs the current position using a given set of moves specified in the class

Type:

class instance
