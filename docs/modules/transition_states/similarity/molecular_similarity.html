Molecular similarity

Module that contains the classes for evaluating similarity of atomic or molecular conformations

class topsearch.similarity.molecular_similarity.MolecularSimilarity(distance_criterion: float, energy_criterion: float, weighted: bool = False, allow_inversion: bool = False)

Description

Separate class to deal with the extra complexity of atomic systems. They require translation, rotation, centering and permutation

distance_criterion

The distance under which two minima are considered the same, for proportional_distance set to True this is not an absolute value, but the proportion of the total function range

Type:

float

energy_criterion

The value that the difference in function values must be below to be considered the same

Type:

float

weighted

Flag that specifies whether we include the relative atomic weights in alignment

Type:

bool
